# FreiFANG – Startstruktur (Jekyll + Minimal Mistakes)

Schnellstart:

1. **Repository clonen** oder diese Struktur hochladen
2. GitHub: Settings → *Pages* → Source: `GitHub Actions` **oder** `Deploy from a branch` (main)
3. Domain (optional) setzen
4. Inhalte in `_posts/`, `_pages/` und `assets/downloads/` pflegen

## Lokal testen

```bash
bundle install
bundle exec jekyll serve
```

Öffne dann: <http://localhost:4000>

## YouTube & Downloads
- YouTube via `<iframe>` wie in `lehrvideos.md`
- Dateien nach `assets/downloads/` legen und verlinken
